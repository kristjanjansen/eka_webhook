## EKA Webhook

Incoming and outgoing webhook enpoint.

Part on the [Estonian Academy of Arts](https://www.artun.ee/en/home/) Emerging Themes masters project.

### Demo

Post something to

https://eka-webhook.now.sh/webhook

### Installation and running locally

```sh
npm i
npm run start
```

### Deploying

```sh
npm i -g now
now
now alias
```
